# ghjournal プライバシーポリシー

**最終更新日: 2026年1月3日**

## 概要

ghjournal は、GitHub リポジトリと同期できる日記アプリです。このプライバシーポリシーでは、アプリがデータをどのように扱うかを説明します。

## データの収集と保存

### 端末内に保存されるデータ

以下のデータはお使いの端末内にのみ保存されます：

- **日記**: 内容、タイトル、日時
- **位置情報**（任意）: 日記に添付する位置情報
- **天気情報**（任意）: 日記用に取得した天気データ
- **GitHub 認証情報**: 端末内に安全に保存

### GitHub に同期されるデータ

GitHub 同期を有効にすると、日記の内容（日付、位置情報、天気を含む）は連携された GitHub アカウントのリポジトリに保存されます。

**重要**: データは連携された GitHub アカウントのリポジトリに保存され、開発者が運営するサーバーには保存されません。

## サードパーティサービス

### GitHub

アプリは以下の目的でのみ GitHub を使用します：

- GitHub アカウントでの認証
- 連携されたリポジトリへのファイルの読み書き

GitHub の認証情報およびデータは [GitHub のプライバシーポリシー](https://docs.github.com/ja/site-policy/privacy-policies/github-general-privacy-statement)に準拠します。

### Open-Meteo（天気情報）

アプリは [Open-Meteo](https://open-meteo.com/) から天気情報を取得します。天気機能を有効にすると、以下のデータがOpen-Meteoに送信されます：

- 位置情報（緯度・経度）
- 日付

Open-Meteoから取得した天気データは [CC BY 4.0 ライセンス](https://creativecommons.org/licenses/by/4.0/) に基づいて提供されます。Open-Meteoのプライバシーポリシーについては [Open-Meteo Terms](https://open-meteo.com/en/terms) をご確認ください。

## 収集しないデータ

- 分析データや利用状況データは収集しません
- 行動追跡は行いません
- 第三者へのデータ販売・共有は行いません
- 開発者のサーバーにデータを保存しません

## データの管理

- **端末内のデータ**: アプリをアンインストールすることで全てのデータを削除できます
- **GitHub データ**: 連携されたリポジトリはご自身の GitHub アカウントにあり、いつでも削除できます
- **位置情報**: 位置情報の利用は任意であり、端末の設定で無効にできます
- **GitHub 同期**: 同期は任意であり、アプリはオフラインでも動作します

## ポリシーの変更

このプライバシーポリシーは随時更新されることがあります。変更があった場合は「最終更新日」に反映されます。
